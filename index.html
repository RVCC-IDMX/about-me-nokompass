The keen-eyed among you may have noticed something strange about the photo gallery.
Just kidding. There is nothing strange, it's just a hover effect! The photos in the gallery above have CSS that makes them scale up when you hover over them! Let me show you howthat works.

First, let's take a look how the HTML is layed out. The first element we have is the section container. This defines the entire area that the gallery can be placed inside. Next is a div with the 'gallerybackground' class. This div is used to achieve the blur effect that you see in every section of this webpage. Next is the h2, which is the title of this section, 'Photo Gallery'. Now we get to the good stuff. There is a section with class of 'galleryimages'. This class contains all images in the gallery, the purpose of this you will see later. Then there is a div with the 'image' class. This class is used on the container for every image, that way we only have to write the style information once! Finally there is an img tag and a div tag. The div is used to store the photo caption and has the class 'desc' (because I forgot what a caption was called when I wrote this code).

Now let's look at the CSS. We start with some basic gallery styling that defines a background image, position, size, etc. We set a size and alignment for the h2, and then we reset the margins for the h2 because by default, it has an ugly margin on it that can mess up your calculations if you don't remove it. Then for the gallery images container we assign a margin and display the images as flexboxes, justifying with space around. The space around allows the webpage to be scaled, and the images to keep an even amount of space between them! Then for the gallery background, we add some padding, a transparent background color using hsl, a box shadow, and a backdrop blur!

Now onto the good stuff. For the image class, we assign a margin and set a width. We display the image container as flexbox because if you remember, we have the images AND the image caption inside of this container. This is so when we set the flex direction to column, it's really easy to align the caption center and allows for an easier animation! Then there is a transition property which sets the transition between the states of the width. The width on the hover state is set to 440px, so when you hover over an image, it transitions to the hovered state! How cool! And finally, there is a width and height property for the img elements, and the image caption gets a 1em margin.
I hope you learned something from this post!
